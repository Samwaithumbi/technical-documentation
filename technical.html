Operators

Operators are used for mathematical and logical calculations

Types of operators

  1. Arithmetic Operators

    2. The purpose of this operator is to perform mathematical operations like addition, subtraction, division, multiplication etc.

  2. Increment Operator

    3. Mainly used for incrementing the value of an integer. It is represented by the ‘++’ operator. ++x will increase the value of the variable x instantly.

  3. Decrement Operator

    4. Mainly used for decrementing the value of an integer. It is represented by the ‘–’ operator. -x will decrease the value of the variable x instantly.

  4. Assignment Operator

    5. This operator is used to assign value to a variable

  5. Logical Operator

    6. Logical operator is used in decision making statements where it returns a 0 for false and 1 for true

                        Operator        What it does
                 && (Logical AND) True only if all conditions satisfy.
                 || (Logical OR) True only if either one condition satisfies.
                 ! (Logical Not) True only if the operand is 0.
  6. Conditional Operator

    7. Also known as Ternary operator and it is also used in decision making statements

    Example of Arithmetic Operator

                        
                        #include 
                            main()
                            {
                            int i = 17;
                            char c = 'c'; /* ascii value is 99 */
                            float sum;
                            sum = i + c;
                            printf("Value of sum : %f\n", sum );
                            }

  1. Library functions are those functions which are already defined in C library, example printf(), scanf(), strcat() et
  2. A User-defined functions These functions are made for code reusability and for saving time and space

Example of function

                        ret = max(a, b);
                        printf( "Max value is : %d\n", ret );
                        return 0;
                        }
                        /* function returning the max between two numbers */
                        int max(int num1, int num2)
                        {
                        /* local variable declaration */
                        int result;
                        if (num1 > num2)
                        result = num1;
                        else
                        result = num2;
                        return result;}
